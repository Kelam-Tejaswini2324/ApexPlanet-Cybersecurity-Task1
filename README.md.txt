# Task 1 - Foundation & Environment Setup

## Objective

The objective of this task is to build a strong foundation in cybersecurity, networking, cryptography, and Linux fundamentals while setting up a basic cybersecurity lab environment.

## Topics Covered

### Cybersecurity Basics

* CIA Triad
* Common Threats
* Attack Vectors

### Linux Fundamentals

* pwd
* ls
* cd
* mkdir
* touch

### Networking Basics

* IP Addressing
* Network Interfaces
* Connectivity Testing

### Cryptography Basics

* Symmetric Encryption
* Asymmetric Encryption
* Hashing
* SSL/TLS

### Tool Familiarization

* Nmap
* Wireshark
* Netcat

## Practical Activities

* Created Linux directories and files
* Practiced Linux commands
* Checked network configuration using ip a
* Tested internet connectivity using ping
* Installed and verified Nmap
* Installed and verified Wireshark
* Verified Netcat functionality

## Learning Outcome

Successfully gained foundational knowledge of cybersecurity concepts, Linux operations, networking fundamentals, cryptography basics, and security tools.
