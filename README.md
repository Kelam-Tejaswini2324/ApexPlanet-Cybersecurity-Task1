# ApexPlanet-Cybersecurity-Task1
Task 1 of the ApexPlanet Cybersecurity &amp; Ethical Hacking Internship Program covering cybersecurity fundamentals, Linux basics, networking concepts, cryptography, and security tool familiarization using Nmap, Wireshark, and Netcat.

## Foundation & Environment Setup

### Objective

Build a strong foundation in cybersecurity, networking, cryptography, and Linux fundamentals while setting up a basic security lab environment.

### Cybersecurity Basics

* CIA Triad (Confidentiality, Integrity, Availability)
* Common Threats (Phishing, Malware, DDoS, SQL Injection, Ransomware)
* Attack Vectors (Social Engineering, Insider Threats, Wireless Attacks)

### Linux Fundamentals Practiced

* pwd
* ls
* cd
* mkdir
* touch

### Networking Basics

* Network Interface Information using `ip a`
* Connectivity Testing using `ping`

### Tool Familiarization

* Nmap
* Wireshark
* Netcat

### Practical Work Completed

* Created Linux directories and files
* Practiced Linux commands
* Verified network connectivity
* Installed and verified Nmap
* Installed and verified Wireshark
* Verified Netcat functionality
